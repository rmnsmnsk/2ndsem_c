#команды

find build -name "hw_1" -type f - пример поиска файла

##форматирование кода

- `fmt` — исправить форматирование всех .c и .h файлов
- `fmt-check` — проверить форматирование

##запуск программ

# Из корня проекта , понятно что всто hw_1 любая другая программа
cmake --build build --target hw_1         # собрать программу
cmake --build build --target hw_1_tests   # собрать тесты
cmake --build build --target check_hw_1   # собрать и запустить тесты

# Запустить программу руками:
./build/hw_1/hw_1
../../build/src/hw_1/hw_1 - из папки с исходниками
